# fractal-openapi

JSON Schema and OpenAPI 3.x support for Haskell with pluggable code generation.

## Features (Planned)

- **Complete JSON Schema Support**: All versions (draft-04 through 2020-12)
- **Vocabulary System**: Extensible vocabulary support for custom keywords
- **Validation**: High-performance schema validation with detailed error reporting
- **Metadata Extraction**: Rich introspection capabilities
- **Pluggable Codegen**: Template Haskell-based code generation
- **Type Generation**: Generate Haskell datatypes with Aeson instances
- **Semantic Newtypes**: Support schema annotations for generating semantic newtypes
- **OpenAPI Support**: Full OpenAPI 3.x specification support
- **Framework Integration**: Generate Servant and Yesod routing types
